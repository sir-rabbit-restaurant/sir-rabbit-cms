= CMS для сайта Сэр Кролик

== Инструкции по запуску

Внесите следующие данные в файл `.env`:

[source,bash]
----
DATABASE_USER=<user>
DATABASE_PASSWORD=<your_password_here>
DIRECTUS_SECRET=<your_secret_here>
----

[source,shell]
----
docker compose up -d
----

== Migration
Для выполнения миграции выполните следующую команду:

[source,shell]
----
docker compose up directus-migrator
----

== Snapshot
Для  создания снапшота выполните следующую команду:

[source,shell]
----
chmod 0777 ./snapshot/* # разрешение на запись в директорию snapshot
docker compose up directus-snapshot-creator
docker compose up --force-recreate directus-snapshot-creator # при проблемах с несуществующей сетью
----
